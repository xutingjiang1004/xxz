# 项目启动与访问说明
# 简易朋友圈（Vercel Serverless 版）

## 一、环境要求
- Node.js 16.x 及以上版本
- 已安装npm/yarn/pnpm包管理器
功能：
- 登录用户可发布文字动态
- 游客可评论、点赞
- 浏览计数与最近浏览用户公开可见
- 数据仅存储在内存中（不使用文件系统/数据库）
